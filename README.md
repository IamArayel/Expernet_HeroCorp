# Expernet_HeroCorp
```
Mettre en place un système permettant de gérer des super héros (la modélisation est à faire). Le système permet de créer, lister, modifier, effacer des héros.
Lors de la récupération  de la liste des héros vous devez avoir la possibilité de filtrer via une zone de recherche par nom, ou par capacité.
Le projet doit être versionné, bootstrappé, et utiliser la POO.

BONUS : L'utilisateur peut exporter la liste sous un format de fichier (pdf, csv, xlsx).
```
